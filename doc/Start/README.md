## 配置Git
    1.创建一个github账户
    2.在本机上设置邮箱和用户名
    3.生成本机的SSH key
    4.将SSH key添加到github

## 检出仓库
    1、将功成fork一份到自己账号下
	2、打开命令行窗口，输入 git clone 复制的地址
	3、git命令执行结束后，在当前目录下出现了github上的工程项目源码

## 修改内容
    1、创建并切换分支
    2、修改内容
    3、add及commit
    4、合并到主分支
    5、push
